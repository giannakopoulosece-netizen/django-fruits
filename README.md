# Django Fruits Project !!

A simple Django web application that demonstrates:
- CSRF-protected forms
- Dynamic routing
- Static images handling
- Basic Django project structure

The user enters their name, selects a fruit, and sees the corresponding image.

---

## Features

- Home page with POST form (CSRF protected)
- Fruit selection page
- Dynamic fruit detail page
- Static images served via Django
- Clean and simple Django architecture

---

## Technologies Used

- Python 
- Django
- HTML5
- CSS
